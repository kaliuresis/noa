This website displays the locations of all the orbs in Noita, including locations to spawn orb 11 from a great chest.

The Great Chest orb can be obtained by getting a great chest to spawn at exactly the given coordinates. You can align an EoE cast at the pixel found by locking yourself in the approximate location, then checking your save00/player.xml.

If the Find Sampo option is turned on, it will search for great chest locations that spawn a sampo instead of an orb.

The farland button sets the search to locations where coordinate rounding allows for a larger than 1-pixel region to align the chest. Thanks to pudy248 for adding this feature.
